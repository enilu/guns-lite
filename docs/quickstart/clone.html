# 克隆本项目

本项目地址为：https://github.com/enilu/guns-lite,如果对你有用，欢迎给个star

项目共三个分支分别为：

  • master 项目主分支，mysql数据库
  • oracle 正如其名，和主分支的唯一区别在于数据库使用oracle，
  • admin-flat 该分支将页面更改为扁平化风格，去掉iframe+标签页的形式

进入控制台输入以下命令将项目克隆岛本地：

git clone https://github.com/enilu/guns-lite.git
